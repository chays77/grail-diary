# The Sticky-Notes Pen

This is NOT a journal. It is a visible, searchable **"not yet filed"** pile — the place
for jots and scraps that don't belong to a focused journal yet.

- One file per sticky: `YYYY-MM-DD-slug.md`, minimal frontmatter
  (`captured: YYYY-MM-DD`, `origin: [where it came from]`), raw body.
- Nothing auto-files into or out of this folder.
- It is drained by Entry Point D (Migration Surfacing — see `grail-capture.md`):
  the agent walks each sticky against a target journal's criteria, you say
  **move / skip**. Moved notes leave the pen; skipped notes stay surfaceable
  on a future walk.
- Nothing here is ever deleted by the agent.
